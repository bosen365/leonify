# leonify
Leon package for Spotify

## Modules

### Play

Play a song, album, artist or playlist

#### Usage

```
"Play God's Plan"
"Play artist Drake"
"Play album Scorpion"
"Play playlist RapCaviar"
```

### Pause

Pause current track

#### Usage

```
"Pause"
```

### Next

Skip to next song

#### Usage

```
"Next song"
```

### Previous

Skip to previous song

#### Usage

```
"previous song"
```

### Repeat

Toggle repeat mode

#### Usage

```
"repeat"
"repeat off"
```

### Shuffle

Toggle shuffle mode

#### Usage

```
"shuffle"
"shuffle off"
```

## Authorization

```
"spotify login"
```
